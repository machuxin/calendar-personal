# 私人日历小助手
## 一个简单、易用的小日历
![image](https://github.com/machuxin/calendar-personal/assets/26918529/8a20bb80-d713-4ba9-86e8-cd4192a5922f)
## 功能简介
* 支持阴历、阳历
* 支持法定节假日、调休展示
* 支持个人日程设置
* 数据保存在本地、0上传 保护个人隐私。
